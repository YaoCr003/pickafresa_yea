REALSENSE-MACOS COMPATIBILITY GUARD

WARNING!
The files within the /realsense_macos package are all scripts that must be run SUDO.
Use tools with caution.
Vulnerabilities have not been found but the scripts can be modified by a 3rd party and make your device vulnerable.
Use at your own discretion. Only use scripts available as official releases.

Remember to uninstall guard after use to avoid conflicts.

Scripts fully tested with the following device/software specs:
macOS 26.0, 26.1. (TAHOE)
Apple Silicon (arm64) M1 Pro
pyrealsense2-macosx 2.54.2
Intel Realsense D435 Camera
